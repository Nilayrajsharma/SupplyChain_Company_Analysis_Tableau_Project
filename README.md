# SupplyChain_Company_Analysis_Tableau_Project

Overall Aim of the Project

The aim of this project is to analyze supply chain operations using Tableau dashboards to monitor inventory levels, order flow, and delivery performance, identify demand patterns and operational bottlenecks, and improve overall supply chain efficiency through data-driven decision-making.


Dashboard 1 – Inventory Analysis
Aim

To analyze inventory levels across products and locations in order to identify overstocking and stock-out situations. This dashboard helps optimize inventory management and reduce holding and shortage costs.

Focus Areas:

Current stock levels

Inventory distribution

Overstock vs understock analysis


Dashboard 2 – Order & Demand Analysis
Aim

To analyze order trends and customer demand patterns over time. This dashboard helps understand high-demand products, seasonal trends, and order volume behavior to improve demand forecasting and planning.

Focus Areas:

Order volume trends

Product-wise demand

Time-based demand patterns


Dashboard 3 – Delivery & Performance Analysis
Aim

To evaluate delivery performance by analyzing delivery time, delays, and order fulfillment efficiency. This dashboard helps identify logistics bottlenecks and improve on-time delivery rates.

Focus Areas:

Delivery time analysis

On-time vs delayed deliveries

Logistics efficiency metrics
